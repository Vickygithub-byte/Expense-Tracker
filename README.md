# Expense-Tracker

This is a simple mvc web application created by me.
I have used .net MVC, Microsoft Sql Server, and 
Chrome WEB Browser.

# Tools used:                                     

* Microsoft.EntityFrameworkCore(6.0.5)             
* Microsoft.EntityFrameworkCore.SqlServer(6.0.5)    
* Microsoft.EntityFrameworkCore.Tools(6.0.5)          
* Microsoft.VisualStudio.Web.CodeGeneration.Design(6.0.5)    
* Swashbuckle.AspNetCore(6.3.0)

# How to Create Database and run application?
1.Open tools                                     
2.Nuget Package Manager                          
3.Package Manager Console                          
4.Run Command "create-database"                  
5.Run the application.
